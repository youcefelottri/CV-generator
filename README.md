# حاسبة معدل البكالوريا

هذه الأداة تتيح لك اختيار الشعبة، إدخال درجات المواد، حساب المعدل العام، وعرض كشف النقاط النهائي مع إمكانية تنزيله كملف PDF.

## ملفات المشروع

- `index.html`: صفحة الويب الرئيسية.
- `styles.css`: أنماط العرض.
- `script.js`: منطق التطبيق وحساب المعدل.

## طريقة التشغيل

1. افتح `index.html` في متصفح الويب.
2. اختر الشعبة.
3. أدخل العلامات لكل مادة.
4. اضغط زر "حساب المعدل" لعرض نتيجة كشف النقاط.
5. يمكنك تنزيل النتيجة كملف PDF.
